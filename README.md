# Design for Your Home - Interior Design Website

A modern, professional website for an interior design company featuring elegant design, responsive layouts, and interactive portfolio gallery.

## Features

- **Responsive Design** - Mobile-friendly layouts that adapt to all screen sizes
- **Modern UI** - Clean, sophisticated design with Open Sans typography
- **Portfolio Gallery** - Interactive filterable gallery with category sorting
- **Service Showcase** - Comprehensive service descriptions with imagery
- **Team Profiles** - Meet the designers with professional bios
- **Contact Form** - Easy-to-use contact form for project inquiries
- **Smooth Animations** - Scroll animations and hover effects throughout

## Pages

- **Home** (`index.html`) - Hero section, featured projects, company intro
- **Services** (`services.html`) - 6 detailed service offerings with process overview
- **Portfolio** (`portfolio.html`) - Filterable gallery of 12 design projects
- **About** (`about.html`) - Company story, values, team members, testimonials
- **Contact** (`contact.html`) - Contact form, office information, FAQ section

## Technologies

- HTML5
- CSS3 (Custom styling with CSS Grid and Flexbox)
- JavaScript (Vanilla JS for interactivity)
- Google Fonts (Open Sans)
- Pexels (High-quality imagery)

## Usage

Simply open `index.html` in a web browser to view the website. All pages are interconnected with working navigation links.
